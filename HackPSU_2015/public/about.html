What is StudyLogic? for Canvas


Educational tool used by a student to gauge and evaluate the intensity of each week of a semester.

StudyLogic utilizes the Canvas API to retrieve information about a student's class lecture times (Enrollments API), quiz dates (Quizzes API), and exam dates (Events API).

StudyLogic lets a student input information about their preferred study habits, which will be combined with the API data in order to:
    -Provide a bar graph that visualizes the intensity of each week of the semester with respect to hours spent studying or in class per week.
    -Create a recommended study schedule for the weeks in which there are no tests or quizzes.
    -Create a customized study schedule for a specific week in which there are tests and/or quizzes, with respect to how much extra effort the student has denoted they would like to put into review sessions.

The most efficient way to utilize this web app is at the beginning of a semester, in order to understand early on which weeks will require the most work, and also to establish a realistic study process that fits both the student's personal study habits and schedule.
